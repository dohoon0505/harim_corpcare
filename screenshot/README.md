# 하림그룹 경조화환 시스템 — 화면·기능 안내

하림그룹 계열사 임직원의 경조사(축하·근조) 화환 주문·정산·관리를 위한 B2B 시스템입니다.
아래는 기능 설명에 필요한 전체 화면 캡처입니다. (데스크톱 전용 · 한국어 전용)

- **기업(계열사) 사용자**와 **관리자(admin)** 두 콘솔로 구성됩니다.
- 관리자 로그인: 아이디 `admin` / 비밀번호 `0324`
- 로그인 없이 열람하는 **공개 거래명세서** 링크를 별도로 제공합니다.

---

## 1. 인증

### 1-1. 로그인
![로그인](./01-login.png)

- 좌측 다크 패널: 브랜드 + 헤드카피 + **당일배송 마감 카운트다운**(시간대별로 마감까지 남은 시간 / 도착 예정 안내가 실시간 표시).
- 우측: 아이디·비밀번호 로그인, 계열사 담당자 가입 링크.

### 1-2. 계열사 담당자 가입
![회원가입](./02-register.png)

- 3단계 위저드(계정 설정 → 담당자 정보 → 사업자 정보) + 완료 화면.
- 비밀번호 강도 표시, 단계별 유효성 검사. 등록 즉시 이용 가능(승인 절차 없음).

---

## 2. 기업(계열사) 사용자 콘솔

### 2-1. 경조화환 주문 — 담당자 선택
![주문 담당자 선택](./03-order-select.png)

- 주문을 진행할 담당자를 먼저 선택합니다. 신규 담당자 등록도 가능.

### 2-2. 경조화환 주문 — 주문서 작성
![주문서 작성](./04-order-form.png)

- **간편접수**: 부고장(근조)·청첩장(축하)으로 빠른 주문.
- **상품 선택**(4종 화환) · 배송지 정보 · 즉시배송 · 리본 문구 · 보내는분 선택.
- **배송완료 알림 수신**: 받는분·보내는분·담당자별 알림톡 수신 토글.

### 2-3. 실시간 주문내역
![실시간 주문내역](./05-orders.png)

- 주문현황(접수대기·주문접수·배송완료)·사진 유무·기간·검색 필터.
- 배송완료 주문은 항상 배송 사진을 보유.

### 2-4. 주문 상세
![주문 상세](./06-order-detail.png)

- 행 클릭 시 배송 사진과 주문 상세정보(담당자·주소·상품·금액·상태)를 모달로 확인.

### 2-5. 거래명세서 조회
![거래명세서](./07-invoice.png)

- A4 거래명세서(공급받는자/공급자/거래내역) + PDF 다운로드.
- 조회 기간 변경, 계산서 발급 동의, **로그인 없이 열람 가능한 공개 링크 복사**.

### 2-6. 정산회계 조회
![정산회계](./08-settlement.png)

- 회사 정보 + 월별 정산 내역(계산서 발급/정산 확인 상태 뱃지).

### 2-7. 프로필 저장공간
![프로필 저장공간](./09-profile.png)

- 리본에 쓰는 **프로필(보내는분)** 과 **담당자** 를 저장·관리(추가/수정/삭제).
- 정산·회계 담당자 지정(거래명세서·입금 알림 수신). 브라우저에 영속 저장.

### 2-8. 상품 규격 안내
![상품 규격 안내](./10-products.png)

- 전 계열사 동일 단가의 상품 10종(상세상품·금액·규격) 상시 노출.

### 2-9. 상품 샘플 사진
![상품 샘플](./11-product-sample.png)

- "보기"를 누르면 상품 샘플 사진·금액·규격을 모달로 확인.

## 3. 관리자(admin) 콘솔

### 3-1. 계열사 접속관리
![계열사 접속관리](./12-admin-clients.png)

- 하림그룹 8개 계열사 계정 목록(회사명·접속 아이디·담당자·연락처·상태).
- 승인 절차 없이 신규 등록/수정/삭제. 상태는 활성/정지.

### 3-2. 신규 계열사 등록
![신규 계열사 등록](./17-admin-client-new.png)

- 계정·회사·담당자 정보를 입력해 계열사를 신규 등록(승인 절차 없음).

### 3-3. 계열사 정보 수정 (접속 아이디 수정 가능)
![계열사 정보 수정](./13-admin-client-edit.png)

- 계정·회사·담당자 정보를 수정. **접속 아이디도 직접 수정 가능**.

### 3-4. 계열사 삭제
![계열사 삭제](./18-admin-client-delete.png)

- 삭제 확인 모달. 계정(아이디·비밀번호)·정산·주문 정보까지 함께 삭제.

### 3-5. 계열사 분리정산
![계열사 분리정산](./14-admin-settlement.png)

- 조회 기간(년/월)별 계열사 정산 종합(청구금액·거래명세서·계산서·거래대금).
- 계열사별 총/입금완료/미입금 요약, 공개 링크 복사 및 명세서 PDF 다운로드.

---

## 4. 공개 거래명세서 (로그인 불필요)

### 4-1. 본인 확인 게이트
![공개 명세서 게이트](./15-public-invoice-gate.png)

- 공개 링크(`/invoice/?link=토큰`) 접속 후 **사업자번호로 본인 확인**.

### 4-2. 거래명세서 문서
![공개 명세서 문서](./16-public-invoice.png)

- 확인 후 A4 거래명세서 열람 및 PDF 다운로드.
